## IMAGE OPTIMIZATION TIPS (Optional):
# ------------------------------------
# Before uploading, consider optimizing images for web:
#   - Use tools like TinyPNG or Squoosh
#   - Target size: < 500KB per image
#   - Format: JPG for photos, PNG for graphics
#   - Resolution: Max 1920px wide
